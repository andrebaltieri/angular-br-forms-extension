# angular-br-forms-extension

Diretivas para componentes de formulários utilizados no Brasil como CPF, CNPJ, CEP, RG, Telefone entre outros.

## Diretivas/Pipes
 * CPF
 * RG
 * CNPJ
 * Telefone
 * Email 
 * URL
 * CEP (Com auto-complete)
 * Conversão Maiúsculo
 * Conversão Minúsculo
 * Camel Case
 * Somente Números
 * Máscaras Customizáveis
